## 🎯 What It Does

### **Wallet-Based Authentication**
```
User → Connect MetaMask → Sign Message → Get Token → Logged In
```

### **Global Auth State**
```typescript
const { user, token, isAuthenticated, logout } = useAuth();
// Available in ANY component
```

### **Protected Routes**
```typescript
<Route 
  path="/dashboard" 
  element={<ProtectedRoute><Dashboard /></ProtectedRoute>}
/>
```

### **Persistent Sessions**
```
Login → Token Saved to localStorage → Refresh Page → Still Logged In
```
